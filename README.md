# DuberyMNL

AI-powered marketing workflow for generating **Facebook ad creatives and marketing assets** for DuberyMNL Polarized sunglasses brand.

This project explores how agent-based automation can assist a small e-commerce business by generating captions, transforming them into creative prompts, and producing ad-ready visuals through a structured workflow.

The system begins as a marketing tool for a real brand and is designed to evolve into a reusable **AI marketing engine**.

---

## Overview

DuberyMNL automates the **creative portion of the marketing process** for Facebook campaigns.

Instead of manually creating captions, prompts, and ad creatives, the system uses specialized workflows to generate these assets automatically.

The automation pipeline converts marketing ideas into visual assets that can be used directly in advertising campaigns.

---

This project aims to:

---

## Current System

The project currently focuses on the **creative generation pipeline** for Facebook marketing assets.

### Caption Generation

Generates Facebook-style marketing captions based on product context, promotional offers, and target audience.

### Caption → Creative Prompt

Transforms generated captions into structured prompts suitable for image generation.

### Image Generation

Produces advertising visuals designed specifically for Facebook ad formats.

The output is a set of **ready-to-use ad creatives**.
