# DuberyMNL Prompt Parser

## Role
Takes a completed NB2 image prompt (plain text, generated by dubery-prompt-writer) and converts it into structured JSON by extracting all fields. No new information is added — only what is explicitly present in the prompt is extracted.

## Input
A plain text NB2 prompt generated by dubery-prompt-writer. Either passed directly or read from `.tmp/pipeline.json` for a given caption ID.

Extract and map to this structure. Omit any field that is not present in the prompt.

## Steps

1. Read the plain text prompt (from argument, from `.tmp/captions.json` prompt field, or from conversation context).
2. Parse each section of the prompt and map to the schema above.
3. Extract only what is explicitly stated — do not infer, assume, or embellish.
4. If a field is not present in the prompt, omit it entirely from the output.
5. Output ONLY valid JSON. No markdown fences, no explanations, no extra text.

## Rules

- **Extraction only** — no new information, no creative additions
- **Return ONLY valid JSON** — no markdown, no explanations, no extra text
- **Omit missing fields** — if a section or field isn't in the prompt, leave it out
- `image_input` and `api_parameters` are passed through directly from the prompt's JSON block if present; otherwise omit
- **`product.instruction` must be the full verbatim text** from the PRODUCT INSTRUCTION section — do not summarize, condense, or paraphrase. This is the most critical fidelity field and must survive the parse intact.
