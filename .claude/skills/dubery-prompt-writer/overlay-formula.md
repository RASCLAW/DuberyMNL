# DuberyMNL Overlay Design Formula
## Learned from 8 Approved Ad Images

Apply these rules when specifying overlays in image generation prompts.
Every rule is observed from approved ads — not invented.

---

## RULE 1: Badge Color = Lens Accent Color

The price badge accent color is ALWAYS derived from the featured product's lens color.
This is the single most consistent pattern across all approved ads.

| Lens | Badge Color |
|---|---|
| Classic Blue mirrored | Teal / cyan |
| Rasta Red / Gold mirrored | Gold / amber |
| Purple mirrored | Purple |
| Brown / dark tinted | Orange-brown or warm gold |
| Outback Series (multi-color) | Orange (series identity color) |
| Scene-mood-driven (sunset, golden hour) | Gradient matching scene palette |

Never invent a badge color. Pull it from the lens.

---

## RULE 2: Badge Shape = Concept Energy

Shape is earned by the concept. Observed shapes in approved ads:

| Shape | Energy | Use When |
|---|---|---|
| Fully-rounded pill (stadium) | Clean, everyday, lifestyle | Default for lifestyle / person shots |
| Circle with border ring | Premium, pride, community | Emotional or community-driven concepts |
| Comic speech bubble | Humor, shock, contrast | Before/after or problem/solution |
| Stacked editorial pills (2 layers) | Catalog, series, retail | Product series or multi-SKU shots |
| Wide rounded rect, 2-line text | Bundle deal, value | ₱1,200 two-pair deals |
| Gradient rounded rect | Atmospheric, mood-driven | Scene-tied concepts (sunset, golden hour) |

**Pills are the most common approved shape.** They are the correct default for clean
lifestyle concepts. Other shapes replace pills only when the concept specifically demands it.
Always name the shape and connect it to the concept energy — never leave it implicit.

---

## RULE 3: POLARIZED Label — 6 Approved Treatments

Never place POLARIZED randomly. Choose the treatment that fits the composition:

| Treatment | When to Use |
|---|---|
| Vertical rotated text, left edge (film-label style) | Price badge is prominent — POLARIZED is a subtle accent |
| Text label directly above the price badge | Informational stacking: reads "POLARIZED → ₱699" |
| Fused inside price badge, stacked with ₱699 | Single combined badge carries both messages |
| Separate smaller pill stacked below price badge | Editorial stacking — product/series shots |
| Floating badge anchored below a product cutout image | Tied to product float, not the person |
| Standalone neutral pill floating mid-frame | When price badge is gradient/colorful and needs separation |

---

## RULE 4: Delivery Zone — Two Styles

| Style | When |
|---|---|
| Full-width footer bar (white or dark, spans entire bottom edge) | Product shots, driving scenes, catalog feel |
| Corner text or floating text (no bar) | Lifestyle / person shots — cleaner composition |

---

## RULE 5: Headline Typography Follows Vibe

| Vibe | Typography Treatment |
|---|---|
| Street / market / everyday | Bold yellow or white, all caps, center-aligned |
| Community / pride | Gold or warm-toned display weight, warm palette |
| Before/after / humor | Two distinct treatments: chalk/rough left panel + bold black right panel |
| Series / product editorial | Bold black, large, hard left-aligned, dark text on light scene area |
| Active / glow-up / fitness | Bold white, right-aligned |
| Road trip / adventure | Bold white, left-aligned, multi-line stack |
| Couple / casual deal | Bold white, friendly casual weight |
| Sunset / golden hour | Bold white, centered, two balanced equal lines |

---

## RULE 6: Logo Placement Mirrors Headline Position

| Logo Placement | Trigger |
|---|---|
| Top right | Headline is center or left-heavy |
| Top left | Headline is right-aligned |
| Top center | Headline is centered |
| NEVER bottom | Bottom zone belongs to delivery and price only |

Logo never overlaps the subject's face. Always respects edge breathing room.

---

## RULE 7: Composition Zones

```
┌──────────────────────────────────┐
│  [LOGO]               [LOGO]     │  ← Top zone: logo always here
│                                  │
│      HEADLINE TEXT               │  ← Upper 40%: dominant headline
│      Body / subhead              │
│                                  │
│  [PERSON / PRODUCT]              │  ← Middle 50%: visual subject
│  [POLARIZED float]   [₱ BADGE]  │  ← Mid-right float zone
│                                  │
│  [DELIVERY]         [PRICE/COD]  │  ← Bottom zone: always delivery + price
└──────────────────────────────────┘
```

Nothing overlaps the subject's face or key product detail.
Overlays frame the perimeter or float in clear empty space.

---

## RULE 8: Special Format Triggers

| Format | Use Only When |
|---|---|
| Split screen diagonal | Before/after or direct contrast concept |
| Floating product cutout image | Bundle deal — shows what you're buying |
| COD hang-tag sticker | Product series / catalog shots |
| Messenger icon | Community or DM-CTA driven concepts |
