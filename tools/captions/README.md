# captions — WF1 caption review: local browser UI + email notification

**What it does**
- Serves a local Flask UI for reviewing AI-generated captions stored in `.tmp/pipeline.json` (PENDING status only).
- Lets you rate each caption 1-5 stars, edit text and hashtags inline, toggle the visual anchor (PERSON/PRODUCT), pick recommended products, and add image-direction notes per caption.
- On submit: writes APPROVED/REJECTED status back to `.tmp/pipeline.json`, moves rejected items to `.tmp/rejected_captions.json`, and saves batch feedback to `.tmp/feedback.json`. Server shuts down automatically after submit.
- Sends a Gmail notification email announcing a new review batch is ready, with caption count, vibes, and the review URL.

**Key files**

| File | What it does |
|------|-------------|
| `review_server.py` | Flask review UI at `localhost:5000`; reads `.tmp/pipeline.json`, writes approve/reject decisions back, shuts down on submit. |
| `send_review_email.py` | Sends a Gmail SMTP notification email with caption count, vibes, and review URL. |

**Run**

```bash
# Start the review UI (run WF1 first to populate .tmp/pipeline.json)
python tools/captions/review_server.py

# Send a review-ready notification email
python tools/captions/send_review_email.py --count 25 --vibes "Commuter, Urban, Sale/Urgency"

# With custom review URL
python tools/captions/send_review_email.py --count 25 --vibes "Commuter, Urban" --url "http://192.168.1.5:5000"
```

**Inputs / outputs**

| Direction | What |
|-----------|------|
| Reads | `.tmp/pipeline.json` (PENDING captions generated by WF1) |
| Writes | `.tmp/pipeline.json` (status, rating, edits per caption) |
| Writes | `.tmp/rejected_captions.json` (captions rated 1-2 stars) |
| Writes | `.tmp/pipeline.json.bak` (backup before each write) |
| Writes | `.tmp/feedback.json` (batch-level feedback text, appended) |
| Email | Gmail SMTP → `REVIEW_EMAIL_RECIPIENT` |

**Auth / env**

| Var | Used by |
|-----|---------|
| `GMAIL_SENDER` | `send_review_email.py` — From address |
| `GMAIL_APP_PASSWORD` | `send_review_email.py` — Gmail App Password (not account password) |
| `REVIEW_EMAIL_RECIPIENT` | `send_review_email.py` — To address |

Both scripts load `.env` from the project root automatically.

**Gotchas**
- `review_server.py` exits with code 1 immediately if `.tmp/pipeline.json` does not exist — run WF1 first.
- The server calls `os._exit(0)` 1.5 s after a successful submit, closing itself automatically.
- Rating threshold: 3+ stars = APPROVED, 1-2 stars = REJECTED.
